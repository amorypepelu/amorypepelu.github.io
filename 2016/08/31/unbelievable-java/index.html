对编译时常量的理解

结果是相等。”abc”、”ab”、”c”在Java里都是String类型的编译时常量。当+运算符的左右两个操作数都是编译时常量时，这个+表达式也会被认为是编译时常量表达式。内容相同的String类型编译时常量会被intern为同一个对象，所以a与b都引用了这个对象，要检查它们是否引用相等，自然得到true。
