静态方法不存在任何静态分派机制

发表于 2017-01-03 分类于 Java

输出

静态方法不存在任何静态分派机制，当一个程序调用静态方法、字段时，要被调用的方法、字段都是在编译时刻被选定的，而这种选定是基于修饰符的编译期类型而做出的，修饰符的编译期类型就是我们给出的方法调用表达式原点左边的名字。

不要用类实例来调用静态方法。
